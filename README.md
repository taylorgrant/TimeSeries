# arfimaSim
Creates fractionally integrated (p,d,q) process of length n

```

# Examples #

### ARFIMA (0,d,0)

#
### ARFIMA (0,d,1)
# ma.df <- arfimaSim(500, .75, 0, .4)

### ARFIMA (1,d,1)
# arma.df <- arfimaSim(500, .75, .4, .4)
```
